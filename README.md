# Gameboy_Project

## Modo de uso:

Execute o programa no terminal e passe o caminho para a rom como argumento. Exemplo no Linux:

```
./executavel /caminho/para/rom/rom.gb
```

## Controles:

- UP: `W`
- LEFT: `A`
- DOWN: `S`
- RIGHt: `D`
- B: `N`
- A: `M`
- START: `O`
- SELECT: `P`
- ESC: Fecha o jogo e salva, caso o cartucho tenha suporte a bateria.
