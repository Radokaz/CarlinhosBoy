# Gameboy_Project

## Modo de uso:

O programa já tem uma interface. Basta escolher a rom diretamente pelo botão "Abrir ROM" ou selecionar a pasta em que o programa irá buscar arquivos com a extensão ".gb" para exibir na lista à direita do menu principal.

## Controles:

Por padrão:

- UP: `W`
- LEFT: `A`
- DOWN: `S`
- RIGHt: `D`
- B: `N`
- A: `M`
- START: `O`
- SELECT: `P`
- ESC: Fecha o jogo e salva, caso o cartucho tenha suporte a bateria.
- T: Alterna entre uma paleta verde que emula as cores da tela LCD do dmg original e a paleta preto e branco.
- C: Abre o menu de pausa durante a reprodução do jogo.

Os controles podem ser alterados acima (com exceção do ESC) podem ser alterados tanto no menu principal quanto no menu de pausa.
As teclas '1', '2', '3' e '4' desligam/ativam os respectivos canais de áudio e não devem ser mapeadas, pois são comandos de debugging
